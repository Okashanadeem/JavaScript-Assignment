JAVASCRIPT ASSIGNMENT-1

Rules for naming JS variables

1. Variable names can only contain _underscoreVariable, $dollarSignVariable, anotherVariable123 and camelCaseVariable.
For example $my_1stVariable
2. Variables must begin with a _underscoreVariabl, $dollarSignVariable or camelCaseVariable.
For example $name, _name or name
3. Variable names are case sensitive 4. Variable names should not be JS reserved keywords
